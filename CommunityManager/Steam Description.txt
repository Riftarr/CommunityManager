[p]This mod allows players to modify the student count generated by communities, as well as their tuition. By default victory conditions are adjusted accordingly.[/p]

[p]This is by no means a perfect re-balance of the game, so many of the game features are balanced around an expected number of students. However I believe it introduces enough of a new challenge to make up for it.[/p]

[h1]How to use:[/h1]
[olist]
[*]Subscribe to both 'EduWorks - Core' and 'EduWorks - Mod UI'
[*]Subscribe to this mod.
[*]When you start the game, in the main menu, click on the "Mod Collection" option.
[*]Find all three mods and turn them on in this order:
	[olist]
	[*]Core
	[*]Mod UI
	[*]This mod and any others dependent on the first two
	[/olist]
[*]Press F10 to open the mod menu and change the settings to your preferences.
[*]Click the Save button to apply your settings. You can now dismiss the mod menu by pressing F10 again.
[/olist]

[h3]Note: New save highly recommended, most settings are applied on 'load game save'/'new game start'. However the student multiplier is applied on 'establish admissions office'.[/h3]

[h1]Explanation of the Settings:[/h1]
- Student Base -
Ranges from 1 to 50, sets the  communities natural spawn count to this new base.
Defaults to 20.

- Student Multiplier -
Ranges from 1 to 20, sets the number that the communities base + random is multiplied by.
Defaults to 1.

- Student Random -
Ranges from 0 to half of Student Base^ , sets the number that the communities base is added or substracted with. This generates a random number between this number and the negative of itself, i.e. if random is 3 then the base number is altered by a random int between -3 => 3.
Defaults to 5.

- Enable Student Relation System -
Set on or off (1 or 0) to enable the communities relation system affecting the student spawn limit. Normally the final count of (Student Base + Student Random) * Student Multiplier is then multiplied by your relation score to reflect how many students will spawn this period. When turned off all possible students will spawn for your school.
Defaults to 1 (On)

- Enable Student Batch Recruitment System -
Set on or off (1 or 0) to enable the batch recruitment system. Normally all your available students are slowly spawned over the period until all possible students have spawned. Turning this off causes all possible students to spawn at the beginning of the period with the first wave.
Defaults to 1 (On)

- Tuition Multiplier -
Ranges from 0.01 to 5, sets the number that the communities expected tuition is multiplied by.
Defaults to 1.25 (I originally  set this to 15% but there is a surprising strain on facilities and you need a little boost to maintain a sense of the games original difficulty)

- Enable Student Victory Adjustments -
Set on or off (1 or 0) to enable various adjustments to challenges involving student counts. If you turn this off you could potentially 'win' the game in the first season...
Defaults to 1 (on)

- Reset -
Resets to [b]mod defaults[/b] [i][u]not game defaults[/u][/i]!
